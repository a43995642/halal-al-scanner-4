# Halal Scanner

This project is a mobile-first application built with:
- **React + TypeScript**
- **Vite**
- **Capacitor (Android)**
- **Google Authentication**
- **AI-based product analysis (Gemini)**

## Getting Started

### Prerequisites
- Node.js >= 18
- Java JDK 17 (for Android builds)
- Android Studio (optional, for debugging)

### Installation
```bash
npm install
```

### Development (Web)
```bash
npm run dev
```

### Build Android APK
```bash
npm run build-apk
```
